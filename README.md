## Build

### Clone

- Clone this repo to your local machine (`https://github.com/vikramraja1995/cover-me.git`)

### Setup

> Install all npm modules

```shell
npm install
```

> Compile React JSX to ES5 (bundle.js)
```shell
npm run build
```

> Start the server
```shell
npm start
```
